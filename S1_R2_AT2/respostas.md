# Diferença entre biblioteca e framework

* A biblioteca é tipo uma coleção funções e códigos já prontas para uso. Por um exemplo, quando estamos fazendo uma calculadora e precisa de uma raiz quadrada, é só chamar a função da bibliotaca.

* O Framework é uma estrutura completa. Se você quiser alterar alguma coisa do Framework, é capaz de ter que fazer do zero, já que é uma estrutura pronta. 

* A diferença entre eles é que, enquanto a  biblioteca pode ser chamada quando quiser, ser usada de uma maneira mais livre e você adiciona ela ao seu proejto. Já o Framework é mais fechado, ele decide quando chamar as funções necessárias e você adiciona o seu projeto a ele.
